# MCPD – Overview del Sistema

Il MCPD (Magneto‑Cyclotronic Pulsed Drive) è un sistema di propulsione avanzato basato sulla generazione, estrazione, accelerazione, compressione e collimazione di plasma ad alta energia.

L’architettura ThetaRail integra:

- Reattore spark toroidale
- Testa di tenaglia per estrazione del plasma
- Railway elettrostatica pulsata
- Stadio ciclotrone + RF
- Gate energetici
- Camera di compressione magnetica (MCPC + θ‑pinch)
- Deflettore dome per thrust vectoring
- Ugello magnetico finale

Il sistema è progettato per missioni interplanetarie veloci (es. Terra → Marte in ~45 giorni con 100 MW).
